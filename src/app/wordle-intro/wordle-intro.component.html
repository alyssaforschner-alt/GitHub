Wordle – Projektvorstellung

Ein browserbasiertes Ratespiel: Errate ein geheimes 5‑Buchstaben‑Wort in maximal sechs Versuchen. Dieses Projekt zeigt Frontend‑Logik, sauberes UI und optionales Multiplayer‑Backend.

Gameplay

Pro Versuch gibst du ein 5‑Buchstaben‑Wort ein. Die Kacheln zeigen dir Feedback:

W
O
R
T
E
  • Grün = Buchstabe & Position korrekt
  • Gelb = Buchstabe vorhanden, Position falsch
  • Grau = Buchstabe nicht im Wort

Bewertungs‑Algorithmus (Kurz)

  1. Erst alle grünen Positionen markieren und aus dem Zielwort zählen entfernen.
  2. Dann übrige Buchstaben prüfen: Wenn noch Vorkommen übrig → gelb, sonst grau.
// Pseudocode
function evaluate(guess, target){
  const res = Array(5).fill('bad');
  const counts = {};                      // Häufigkeit im Zielwort
  for (let i=0;i<5;i++) counts[target[i]] = 1 + (counts[target[i]]||0);
  for (let i=0;i<5;i++) if (guess[i]===target[i]) {res[i]='ok'; counts[guess[i]]--;}
  for (let i=0;i<5;i++) if (res[i]==='bad' && (counts[guess[i]]||0)>0){res[i]='warn'; counts[guess[i]]--;}
  return res; // ['ok','bad','warn',...]
}

Installation & Start

  1. Clone/Download Projektordner.
  2. Öffne index.html in VS Code.
  3. Nutze Rechtsklick → Open with Live Server (Extension).

Tipp: Ohne Backend funktioniert alles lokal. Für Multiplayer siehe unten.
